## Responsible-use boundary

This public implementation is a research and marketing-workflow prototype. It is not a medical device, diagnostic service, clinical decision-support system or substitute for medical, legal or regulatory advice. Do not submit patient-identifiable information, protected health information, confidential trial data, private contact databases, credentials or payment-card details.

The agent drafts locally in the browser. It does not scrape contacts, autonomously send outreach or publish unsupported claims. Accountable human review is required before external use.
